rmdir (remove directory) -> удаляет указанную директорию(если нет файлов)
rm (remove) -> удаляет файлы 
pwd (print working directory) ->  выводит путь до текущей директории
touch -> создает указанный файл
rm -rf -> принудительное удаление файлов и директорий
ls -a -> показывает список всех в.т.ч. скрытые файлы/папки
ls -l -> выводит список директорий и файлов с подробной информацией 
ls -la -> комбинация ls -l + ls -a
mv -> переименовать(mv filename.txt newname.txt) или переместить (mv filename.txt ~/путь к директорию ) или переместить в предыдущую директорию mv filename.txt ../
cat -> выводит содержимое указанного файла
nano -> открывает графический интерфейс в терминале для редактирования файла * если нет указанного файла, то создается новый файл с таким именем (nano nameoffile) CTRL+S для сохранения внутри графического редактора 
echo -> команда для быстрой записи данных в файл через терминал (перезаписывает содержимое)
echo text > название файла
cp - copy команда для копирования файла в нужную директорию cp filename directorypath
